## 1. 算法与数据重构
- **过滤与加权**：修改 `douban.js` 中的 `fetchDoubanData`，过滤掉评分 < 4 的项。将年份统计从 `count++` 改为按评分加权（4星+4，5星+5）。
- **轴范围优化**：在 `App.jsx` 中对 `yearData` 进行裁剪，从第一个有有效数据的年份开始展示。

## 2. 图表 UI 深度重组
- **内嵌容器**：将标题、描述和图例包装在一个绝对定位的 `div` 中，置于 `BarChart` 的左上角。
- **样式调整**：将图表区域设为 `relative`，取消外部标题，使图表区成为页面的核心视觉焦点。

## 3. 动态防压盖机制
- **高度预测**：计算前 20% 年份中的最高加权得分。
- **自适应偏移**：如果最高柱子的高度占比超过设定阈值（如 40%），则自动通过 CSS `translate-y` 或增加图表 `top margin` 来抬高文字层，确保不与柱状图重叠。

## 4. 验证与推送
- 验证权重计算是否正确反映在柱子高度上。
- 在不同数据分布（有无早期长尾）下测试布局的自适应能力。
- 推送至 GitHub 触发自动部署。
